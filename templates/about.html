Génération de PDF

Notre projet prend en charge la génération instantanée de rapports PDF. Les utilisateurs peuvent télécharger des données spécifiques ou des ensembles de données entiers au format PDF. Cette fonctionnalité est mise en œuvre à l'aide de la bibliothèque ReportLab, permettant aux utilisateurs d'obtenir des rapports détaillés et formatés pour leurs analyses.

Intégration de la Base de Données

Le projet exploite MongoDB comme base de données backend pour stocker et organiser les ensembles de données. En utilisant la bibliothèque PyMongo, nous établissons une connexion à l'instance MongoDB, facilitant la récupération et la manipulation des données de manière transparente. Ce choix assure une extensibilité et une flexibilité dans le traitement de divers ensembles de données.

Initialisation des Données

Lors de l'initialisation du projet, un fichier Excel contenant des entrées de jeu de données initiales est lu et traité à l'aide de la bibliothèque Pandas. Les données sont ensuite transformées en un format de dictionnaire et insérées dans la collection MongoDB. Cette configuration unique garantit une base de données peuplée dès le lancement du projet.

Présentation du Projet

Notre projet est un système complet de visualisation et de récupération de données développé à l'aide de Flask, un puissant framework web en Python. Le système s'intègre de manière transparente à une base de données MongoDB pour stocker et gérer les ensembles de données de manière efficace. L'objectif est de fournir aux utilisateurs une interface conviviale pour explorer, télécharger et analyser les données de manière pratique.

Interface Utilisateur

L'interface web, propulsée par Flask, offre une expérience utilisateur propre et intuitive. Les utilisateurs peuvent effectuer diverses actions, notamment la recherche d'entrées de données spécifiques, la visualisation d'ensembles de données à travers des graphiques et le téléchargement de données sous forme de tableaux et de fichiers PDF. L'interface est conçue pour être réactive et accessible.

Fonctionnalité de Recherche

Le système intègre une fonctionnalité de recherche robuste permettant aux utilisateurs d'interroger la base de données pour des entrées de données spécifiques. Un motif d'expression régulière insensible à la casse est utilisé pour améliorer la précision de la recherche, et les résultats sont présentés de manière claire sous forme de tableau.

Représentation Graphique

Les utilisateurs ont la possibilité de visualiser les données à travers des graphiques interactifs générés à l'aide de Matplotlib et Seaborn. Le système crée dynamiquement des graphiques linéaires en fonction des attributs de l'ensemble de données, offrant des informations précieuses sur les relations entre différentes variables.
